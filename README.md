# World Layoffs Data Cleaning (SQL)

## Overview
This project focuses on **cleaning a global layoffs dataset using SQL**.  
The goal is to transform raw, inconsistent data into a clean, reliable dataset suitable for analysis.

---

## Dataset
The dataset contains worldwide company layoff records, including:
- Company
- Location
- Industry
- Total laid off
- Percentage laid off
- Date
- Stage
- Country
- Funds raised

> The dataset year is not explicitly defined, which does not affect the data cleaning process.

---

## Data Cleaning Tasks Performed
The following steps were carried out using SQL:

- Removed duplicate records
- Standardized text fields (company, industry, country)
- Trimmed extra spaces and corrected inconsistencies
- Converted the date column to a proper SQL date format
- Handled NULL and blank values
- Removed records with insufficient critical data
